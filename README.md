# README

## Project Overview
This repository contains the custom WordPress theme developed for KC Tennis Blast, created as part of the CP3402 project. Built using the Underscores (_s) starter theme, this project focuses on delivering a clean, modern, and engaging website tailored to promote tennis programs, events, and community participation.

The theme is designed with a vibrant and energetic visual style that reflects the dynamic nature of tennis. A bold colour palette, clear typography, and strong visual hierarchy are used to highlight key information such as upcoming events, coaching programs, and registration opportunities. The layout emphasizes accessibility and ease of navigation to ensure users can quickly find relevant content.

The primary target audience for this theme includes:
- Junior and senior tennis players
- Parents seeking coaching programs for children
- Community members interested in local tennis events

To support these users, the theme incorporates structured content sections, call-to-action elements, and intuitive navigation. Interactive features are used sparingly to maintain clarity and avoid overwhelming users, ensuring the focus remains on engagement and sign-ups.

Overall, the KC Tennis Blast theme aims to provide a professional yet approachable online presence that encourages participation and fosters community involvement in tennis activities.
